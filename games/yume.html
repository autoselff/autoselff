About

YUME is an open-source, multiplatform game engine written from scratch in C++ using OpenGL. The goal of the project is to explore graphics programming and create a lightweight 2D game engine. It also includes support for AILib.

As of now, development on YUME is paused due to a lack of contributors and limited available time. The current version of the engine on GitHub is in the process of being rewritten, aiming for a cleaner architecture and improved modularity.
